Comando para rodar o pytest:
```
pytest              >> Roda os testes
pytest -v           >> Mostra o nome dos teste executados
pytest -s           >> Caso tenha print mostra na saÃ­da
pytest -k "nome"    >> Roda o teste que tenha o nome passado
pytest -x           >> Para os teste caso haja falha
pytest --pdb        >> Abre o debugger
pytest -m           >> Roda o teste com marcado especifico
pytest -rs          >> Mostra o motivo do teste ter skipado
```
